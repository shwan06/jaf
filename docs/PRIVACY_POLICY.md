# Privacy Policy — Maze Rush Runner

_Last updated: 2026-06-27_

This is a template privacy policy for **Maze Rush Runner**. Google Play
requires a publicly hosted privacy policy URL for every app. Host this file
(e.g. GitHub Pages or a Gist), then paste the URL into the Play Console.
**Review and adapt it to what your shipped build actually does** before
publishing.

## Summary

Maze Rush Runner, as published from this repository, is a single-player,
**offline** arcade game. It does **not** collect, store, or transmit any
personal data. There are no accounts, no ads, no analytics, and no network
requests in the base build.

## Information we collect

**None.** The game does not request, gather, or send any personally
identifiable information. Gameplay state (current level, score) lives only in
device memory during play.

## Permissions

The app requests no runtime permissions and declares no sensitive permissions
in its manifest.

## Children's privacy

The game is suitable for all ages and collects no data from anyone, including
children under 13. It complies with COPPA by virtue of collecting nothing.

## Third-party services

The base build integrates no third-party SDKs.
